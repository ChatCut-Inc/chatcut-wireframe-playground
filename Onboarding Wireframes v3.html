Consequences — what changes, what survives

keepMarketing site untouched

  • chatcut.dev — hero, features, testimonials, pricing, blog. Zero SEO impact.
  • "Start free" routes to /signup which lands in the new app home (not the old PromptCreator).
  • Feature demo clips become Examples tiles in the gallery — reused, not rebuilt.

replacePost-auth routes

  • / (today's PromptCreator) → creator + gallery.
  • /projects → merged into gallery via "Your projects" tab.
  • /editor/:id → /p/:id · chat-left / artifact-right.

absorbScenarios & templates

  • 6 scenario cards become sub-chips inside Edit / Generate tabs.
  • TemplateBrowser modal dies → becomes the From template tab.
  • New Examples row in the gallery for runnable inspiration.

navHome button is the spine

  • Top-left Home on every /p/:id. Click → gallery. No breadcrumbs.
  • Deep links to projects work from anywhere.
  • Shared-link collaborators see "Fork into my projects" where Home would be.

killChat-history sidebar (my v2 mistake)

  • Video projects are too heavy for a text-chat history metaphor.
  • Gallery tiles = history, thumbnail-first. Search + filter live there.

dataProject is the unit

  • No schema change to Project in @chatcut/zero.
  • Add: kind, pinned_artifact_id, thumbnail_url, last_accessed_at.

phaseShip order

  • P1 new home behind a flag; create still redirects to old editor.
  • P2 rebuild editor as /p/:id chat+artifact shell.
  • P3 kill TemplateBrowser & /projects page; migrate thumbs.
  • P4 Examples row + power-user layout memory.

metricsWhat this should move

  • Signup → first export: target ↓30%.
  • % new users clicking a creator tab (teaching signal).
  • Day-1 return gallery click-through (thumbnail recall).
  • Template-tab engagement (should jump when first-class).
